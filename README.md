# Steps to run this project on your local computer:

1) Install VS Code on your device from the given link : https://code.visualstudio.com/download

2) Install git from the given link 

   --for windows: https://git-scm.com/download/win

   --for Linux: https://git-scm.com/download/linux


3) Download Node from the given link: https://nodejs.org/en/download




* Open vs code and type following commands in terminal
1. git clone https://github.com/ArnavU/Project-CRS-Frontend.git
2. npm install
3. npm run dev

On localhost://portNo. you will find this web application running
